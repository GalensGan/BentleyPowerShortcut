# Microstation超级快捷键功能使用说明

超级快捷键是一款可以让你在使用 Bentley 的产品时，拥有和 CAD 一样的快捷键体验的插件。

## 🥝使用

1. 单击空格，如果安装成功，会弹出快捷键输入窗体：

   ![](https://i.loli.net/2021/08/24/s9TlypXKogYv4NZ.png)

2. 在里面输入定义的快捷键，按 Enter 或者 空格 响应命令
